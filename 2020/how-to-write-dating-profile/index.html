用做题家的思路写相亲帖

你第一眼看到的是不是也是那张歪了的椅子？

Dec 10, 2020
4 minute read

钱钟书说：「何必要认识那下蛋的母鸡呢？」，但我今天非要写一下我是怎么下蛋的。

写相亲帖之前我也想过如果我真的用「陌上花开体」写一篇相亲帖会是怎么样：

首先大略记录一下自己的成长经历，也就是学习及工作经历，在这其中掺杂一两句无伤大雅故作幽默的俏皮话但重点仍然要落在名声最响亮的那所母校/公司上。然后再罗列一些十分「拿得出手」的爱好，比如运动健身，爬山野营，或莳花弄草，游山玩水，必不可少的肯定是与下厨相关的，可以不是大厨但至少也得爱做饭；当然这其中也得有一些俏皮话，一般以贴标签的形式展现，比如自称吃货/文艺女青年/精致的猪猪女孩/非典型理工女/etc.
在这其中不忘列举一两个故事来侧写一下人物宜家宜室的性格，比如聪明，勤奋，上进，贤惠，开朗，不一而足。

之后是我不知道为什么要有的有关家庭状况的版块。就，也不是什么名门望族联姻，为什么要关心我爹妈在干啥，我爹妈都不关心我在干啥！构思到这里我已经觉得此路不通，但我觉得还可以抢救一下。

然后下一个版块是感情经历。我对感情经历最感兴趣了，哪里有八卦，哪里就有我。结果定睛一看，大家都轻描淡写带过了自己的恋爱 （或者是没有过恋爱） 。总而言之言而总之，从过往的感情中成长了，并且看起来大家都有很体面的分手理由。

最后写对伴侣的期待，看到168的女生期待对方身高170+的时候我沉默了 （是我太肤浅吧） 。其他的条件 （除了地区） 我觉得都称不上条件，起不到多少过滤作用啊，在地铁随便抓一个人的感觉都可以符合，况且地铁上抓到的人极大可能是同城的。有些人对收入水平有要求的，那可以在写字楼里随便抓啊，自己的写字楼抓不到可以去朋友的写字楼里抓呢。我百思不得其解，这个平台怎么会有30%的成功率啊？

（虽然很讨厌网络烂梗，但我还是想说：就这？就这？就这？？？）

但如果不用这一套框架来写，我也有点不知道该从何下手。我回想二十几年来做过的或长或短的自我介绍，尤其是去年和今年，为了找工作密集地自我介绍，总结出做自我介绍的重要法则：说对方想听的。
在工作面试的时候面试官不会在意我小学在哪里读的，而在意我能不能胜任这个职位，所以我的自我介绍应该全方位多角度地展现 （装作） 我足以胜任这份工作。

那么，一篇相亲帖的受众到底是谁？

我感觉很多发帖人没有想清楚这一点，又或者在所谓专业红娘的帮助与修改之后他们已经忘记了受众该是谁。平台的运营者自然觉得受众是这个平台所有的关注者，是点开推送的每一个人。但对于想要找寻伴侣的「嘉宾」来说，受众应该是那个希望与之发展关系/共度一生的人，不管阅读量是几百还是几千甚至十万加，最终结婚的也只有一个人。一篇相亲帖本来就只该写给一个人看，点开推送的人里的绝大多数的意见不重要。

但我也能够理解为什么大家都削足适履把自我介绍写得千篇一律，我把这归结为「做题家追求标准答案的惯性」。就像我也在写之前看了很多别人的帖子（参考答案）才下笔，我们做题家是如此擅长寻找标答里的规律并遵循规律啊。试卷存在标准答案，但爱情存在标准答案吗？婚姻存在标准答案吗？生活存在标准答案吗？即使存在，你想要那一份标准的爱情，想过标准的生活吗？

反正在我看来，生活不存在标准答案的，就算有，我也要反其道而行之 （但反着走是不是另一种「标答」就不得而知了） 。我就想奇奇怪怪地生活。我写完以后也担心自己会因为过于奇形怪状而被骂，但其实一个也没有 （朋友：你这个帖子一看就知道骂不过你） 。倒是收到了一些朋友的喜爱，我膨胀了好几天。

如果回到做题家的体系里，关于「相亲帖该怎么写」的解题思路应该落在「相亲的本质是筛选而不是迎合」。假如真的有一份所谓的标准答案，也应该是发帖人作为出题者，然后在一个平台撒下天罗地网找寻那位高分考生。

之所以今天想起来说这些是因为读了《人物》对梁永安的采访《为什么相爱变得越来越难？》。我感觉有越来越多人已经不想相爱了，不屑追求爱情，觉得这些小情小爱上不得台面，又或者觉得爱情敌不过面包之类的。我想爱情确实是敌不过面包的，在另一个维度上：爱情敌不过功利心。非要说，爱情能够敌过什么呢？我的输入法联想说「爱情敌不过时间」，爱情也敌不过距离，爱情也敌不过自私的人性，它的的确确如小王子的玫瑰一般脆弱。我甚至也不觉得爱情能够敌过孤独，恋爱大约也只是两个人一起孤独罢了。

但我觉得这世界上除了人类之间真挚的感情之外的东西都更加不值得追求，金钱也好，名利也罢，对我而言差不多够用就行了，也就爱值得为之下狠心努力一下了。在这个星球上，我的同类，觉得恋爱大过天的人类，愿意苦心孤诣研究与追求爱情的人类，越来越少了。所以，找寻同类的信号必须比以前更加清晰更加响亮更加有辨识度才行，所以相亲帖也得另辟蹊径才行，你说是不是？
